双目关键点检测 · 使用说明

实时检测针体 N 个关键点的三维坐标与 6D 位姿，并显示帧率

0. 先设好路径变量

打开终端，按你的实际路径粘贴一次（标 ✏️ 处改成自己的）：

DATA=/root/autodl-tmp/data/surgical_seg            # ✏️ 数据目录
UNI=/root/autodl-tmp/code/UniMatch-V2_local        # ✏️ 代码目录
CKPT=$UNI/exp/combined_r100_base/best.pth          # ✏️ 分割模型权重
CALIB=/root/autodl-tmp/code/SAM2-Plus/tools/needle_calib.json   # 标定文件
cd $UNI

1. 三种输入方式（任选其一）

输入在命令里加这些参数
① 采集卡（一个设备，左右并排画面）--capture 0 --layout sbs
② 视频文件（左右各一个）--left 左.mp4 --right 右.mp4
③ 已有数据集序列（按帧回放）--root $DATA --dataset march_1 --key 1_01

采集卡若是上下排列画面用 --layout tb。两台相机也可写成 --left 0 --right 1。

2. 运行（实时显示 + 保存结果）

示例：采集卡输入

python realtime_stereo_keypoints.py \
  --config configs/surgical_combined.yaml --checkpoint $CKPT --calib $CALIB \
  --capture 0 --layout sbs \
  --num-keypoints 5 --seg-size 640 \
  --save-video /root/autodl-tmp/exp/out.mp4 \
  --save-results /root/autodl-tmp/exp/result.jsonl

示例：视频文件输入 — 把上面的 --capture 0 --layout sbs 换成 --left 左.mp4 --right 右.mp4。

示例：数据集回放输入 — 换成 --root $DATA --dataset march_1 --key 1_01。

  • --num-keypoints 5：输出几个等距关键点（可改）。
  • --save-video 文件.mp4：录制可视化（服务器无界面时用这个看效果）。
  • --save-results 文件.jsonl：三种输入都会逐帧保存结果，供后续算指标。
  • --show：弹窗实时显示（仅在有显示器 / VNC 的机器上用；纯命令行服务器会报 Qt 错误，别加）。

3. 屏幕会显示什么

  • 左右并排画面，针体上标出 N 个关键点（针尖绿、针尾红、中间黄）。
  • 左上角：实时帧率 FPS。
  • 下方两行：物体 6D 位姿——平移 t=(x,y,z)mm 与旋转 rot=(x,y,z)deg（欧拉角）。
  • 位姿验证（反投影）：每个视图画出物体坐标轴（X 红 / Y 绿 / Z 蓝），并把三维关键点反投影成白色圈。 白圈若压在检测点上、左右都对齐，说明位姿/三维可信。（不想要可加 --no-reproject）
  • 结果文件：--save-results out.jsonl 存每帧关键点三维坐标 + 完整位姿 {R,t,rvec}； --save-poses out.csv 另存 t / rvec / 旋转矩阵 R(9) / 欧拉角（带表头）。

4. 显存 / 速度调节

需求怎么做
显存不够（要 <4G）调小 --seg-size：640 → 512 → 448
想更快（首选）用 small 模型：--config configs/surgical_combined_small.yaml --checkpoint exp/combined_r100_small/best.pth（约快 3~4×）
想更快（其次）调小 --seg-size（默认已开 fp16 + 左右合批加速）
想更清晰（细针细线）调大 --seg-size：768 / 896
位姿抖动默认已开卡尔曼平滑；如需关闭加 --no-smooth

5. 数据集回放：自动算精度

用 ③ 数据集输入，并以之前生成的关键点作真值，跑完自动打印指标（像素误差 / PCK / 三维误差）：

python realtime_stereo_keypoints.py \
  --config configs/surgical_combined.yaml --checkpoint $CKPT --calib $CALIB \
  --root $DATA --dataset march_1 --key 1_01 \
  --num-keypoints 5 --seg-size 640 \
  --gt-subdir keypoints --pck-thresh 10 \
  --save-video /root/autodl-tmp/exp/verify.mp4 \
  --save-results /root/autodl-tmp/exp/result.jsonl
常见问题
• 弹不出窗口/无界面：需要桌面或 VNC 环境；纯命令行服务器请去掉 --show，改用 --save-video 录像查看。
• 报 CUDA out of memory：把 --seg-size 调小。
• 报找不到文件：检查 $DATA/$CKPT/$CALIB 路径是否填对、是否在同一个终端里设过变量。
